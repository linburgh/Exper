twaver.ShapeLink Class

Extends twaver.Link

它是由一系列的点决定线路走向的一种连线。可以用于展示一些复杂线路的连线。通过添加、删除点可以更改连线的形状

Constructor

twaver.ShapeLink

(
  • [id]
  • [fromNode]
  • [toNode]
)
twaver.ShapeLink

Parameters:

  • [id] Object optional

    折线ID，用于唯一标识折线对象。如果为空，TWaver会按照默认的规则给折线ID赋值。ID可以为string，number类型，可以为Node类型，也可以为Object类型。为string或number类型时指的是折线的ID，为Node类型时，fromNode和toNode相同，都为ID参数传入的Node对象，如果为Object类型，可以传入一个带折线属性的对象，比如： var shapeLink = new twaver.ShapeLink({ name:'shapelinkname', fromNode: twNode, toNode:bbNode });

  • [fromNode] twaver.Node optional

    起始节点

  • [toNode] twaver.Node optional

    结束节点

Returns:

twaver.ShapeLink:

折线对象本身
